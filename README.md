# ArduinoLedController

A simple arduino project to control a rgb ledstrip over http using an ESP8266.

**Libraries** <br/>
ESP8266WebServer

**Parts**
- 1 x ESP8266 Board (I used a NodeMCU)
- 3 x Transistors
- 3 x 1 kohm resistors
- 1 x RGB Ledstrip
- 1 x 12 V Power supply (0.4 - 1 amps per meter)
